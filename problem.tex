A unit sphere packing in dimension n is a set of unit spheres with disjoint interiors.  You can also think of it as a set of points (sphere centers) in n-D space with minimum distance 1.  The tangency graph is the graph taking these spheres as vertices, and tangent pairs (centers at distance one) as edges.  A coloring of the packing assigns a color to each sphere (or its center) such that tangent pairs (centers at distance one) have different colors.  The chromatic number of a packing is the minimum number of colors that can be used in a coloring.  What is the maximum possible chromatic number for a unit sphere packing of dimension n?  Now the upper bound is exponential in n by greedy coloring, the lower bound is almost linear, best record by a work of Hao Chen.  Please improve either bound
